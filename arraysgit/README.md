# Exercises: Arrays
### Beschreibung
Aufgaben um den Umgang mit Arrays erneut zu üben und zu vertiefen.

### Aufgabenstellung
Die folgenden Klassen implementieren:
* `AverageCalculator`
  * Ermittelt den **Durchschnitt** basierend auf den übergebenen Werten  
* `HighScoreIdentifier`
  * Ermittelt den **höchsten Wert** der übergebenen Werte
* `Reverser`
  * Liefert die übergebenen Werte in **umgekehrter Reihenfolge** zurück.
* `SumCalculator`
  * Liefert die **Summe** der übergebenen Werte zurück.

Die Tests zu den einzelnen Klassen sind unter `src\test\java` zu finden.

### Ziel
Die Aufgabe ist erledigt, sobald **kein** Test mehr fehlschlägt.

### Anmerkung
- Die Tests geben Hinweise auf die erwartete Funktionen der einzelnen Methoden.
- Gegebenenfalls Selbststudium notwendig.
